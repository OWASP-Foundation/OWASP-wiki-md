''' 4.12 Client-Side Testing '''

-----

Client-Side testing is concerned with the execution of code on the
client, typically natively within a web browser or browser plugin. The
execution of code on the client-side is distinct from executing on the
server and returning the subsequent content.

The following articles describe how to conduct a Client-Side test of a
web application:
