<td><h2 id="owasp_hive">OWASP HIVE</h2>
<p>OWASP HIVE project is an idea for a learning platform by using some small capable pc to do our bidding.</p>
<h2 id="introduction">Introduction</h2>
<p>The issue - to be a hive we need to connect in some kind of secure anonymous cool way! Using TAHOE-lafs as a storage solution we are able to interlock anything we do into the GRID. Whatâ€™s the GRID? The GRID is a group of Tahoe enabled capable computers (The BeagleBone Black) could be any computer. The idea is to share data across a GRID in a secure anonymous fashion. Using Tahoe-LAFS we can share and distribute project files lots and lots of data all gathered from the GRID.</p>
<h2 id="description">Description</h2>
<p>This learning environment can be used for teaching and proof of concept for projects.</p>
