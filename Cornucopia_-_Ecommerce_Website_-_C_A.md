**Card/Value:** A

### Description:

You have invented a new attack of any type.

### Technical Note:

Players can discuss any type of attack they think might be possible
against the assessment target. It does not matter if the attack relates
to another suit or any other card, but if possible try to identify an
attack that is fairly unique to the application/functionality/users.

### References:

Read more about application security in OWASP’s free Guides on
Requirements, Development, Code Review and Testing, the Cheat Sheet
series, and the Open Software Assurance Maturity Model.
